# Internship Tracker (Java)

## Overview
This is a simple Java console application that allows users to track internship applications and their statuses.

## Features
- Add internship applications (company, role, status)
- View all saved applications
- Menu-based console interaction

## Technologies Used
- Java
- ArrayList
- Scanner

## How to Run
1. Open the project in IntelliJ IDEA
2. Run the `InternshipTracker.java` file
3. Use the menu in the console

## Example Output
Company: Google, Role: SWE Intern, Status: Applied
Company: Amazon, Role: SDE Intern, Status: Interview
